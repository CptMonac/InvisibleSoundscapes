Invisible Soundscapes

This project was created for the Software Structures for User Interfaces (SSUI) final assignment. It was inspired by Julian Treasure's Ted Talk "Why architects need to use their ears".
The google map displayed on the left illustrates noise data collected using my Android phone while walking around the Pittsburgh area. As you can tell, the data is a little sparse, but I think it serves as incentive for crowdsourcing noise data.
